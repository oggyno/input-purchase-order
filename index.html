Catatan Setup:

  1. Buat/buka Google Sheet dan copy ID dari URL
  2. Pastikan sheet "Dataset Purchase Order" sudah ada di spreadsheet
  3. Buat header di sheet tersebut: PO Number | PO Date | Supplier | Item | Nama Barang | Quantity | Description
  4. Di Google Cloud Console, aktifkan Google Sheets API
  5. Buat API Key dan copy ke form di atas
  6. Set sharing Google Sheet ke "Anyone with the link can edit"
